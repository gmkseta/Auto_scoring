## Getting Started

### 1. Install Node js & mongodb

* `sudo apt-get update`
* `sudo apt-get install nodejs mongodb npm`

### 2. Install NPM module

*  `npm install`

### 4. Change compile-run module into custom module

* `cp -r compile-run/ node-modules/ `

### 5. Execute.

* `node index.js`
